Guida Rapida — Controller Campane Chiesa

Passaggi essenziali per uso e diagnostica.

Accesso Rapido

  • AP: SSID ChurchBells-Config, Pwd campanile123
  • UI: http://<IP-dispositivo> (IP sul display/UI)

Comandi Principali

  • Abilita/Disabilita campane: scheda Stato
  • STOP di emergenza: pulsante rosso
  • Test relè: scheda Relè
  • Melodie: scheda Melodie (Riproduci/Stop)
  • Orario: scheda Orario (datetime-local)
  • Resync NTP: scheda Impostazioni

Diagnostica

  • /api/status — stato completo
  • /api/i2c-scan — dispositivi I2C (0x68=RTC)
  • /api/relay-status — livelli pin relè/LED

Aggiornamenti (opz.)

  • Firmware: PlatformIO → Upload
  • UI (SPIFFS): PlatformIO → Upload Filesystem Image

Cablaggio (sintesi)

  • RTC: SDA=21, SCL=22, 3.3V, GND
  • Relè: IN1=25, IN2=26 (active‑low)

Note di Sicurezza

  • Relè active‑low: ON = livello LOW (0)
  • Verifica cablaggi e alimentazioni
  • Cambia password in produzione
